# EmbyReporter
Report issues in Emby.

<img width="422" height="290" alt="{CD969223-3DED-43EF-B523-4BB962DF09B6}" src="https://github.com/user-attachments/assets/a2fc115f-0eb3-492a-a00d-cf39c5e12c12" />

Adds a report issue button to the context menu of videos in Emby allowing users to report issues.

NOTE:
This will requires the plugin to change the internal Emby file actionsheet.js, the plugin will do this automatically when a button is pressed, similarly it will undo this when another button is pressed.
A backup of the file is made for easy manual undo of the change.

WARNING!!!
This is early beta so there might be bugs.
